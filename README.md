# System Design Puzzle

システム設計面接に出てくるようなアーキテクチャを、パズルのピースを組み合わせて構築する Web アプリゲーム。

## コンセプト

- お題が与えられる（例:「100万人が同時視聴する動画配信サービスを設計せよ」）
- パズルのピース（CDN, Load Balancer, DB, Cache, Queue 等）を組み合わせる
- 正解は1つではなく、コスト・可用性・レイテンシのトレードオフがある

## 題材の例

- YouTube
- Discord
- Twitter
- MLOps パイプライン
- PB級データシステム

## 動機

- システム設計面接の勉強
- エンジニア能力向上
- 個人開発・マネタイズ
