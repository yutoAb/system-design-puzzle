プライバシーポリシー

yutoAb（以下「運営者」）は、「システム設計 模擬面接」（以下「本サービス」）におけるユーザー情報の取り扱いについて、以下のとおり定めます。

1. 取得する情報

  1. アカウント情報: Google ログインを通じて取得するメールアドレスおよびユーザー識別子
  2. 取引情報: チケットの購入・消費の履歴（決済に用いたカード番号等は運営者は保持せず、決済代行事業者である Stripe 社が取り扱います）
  3. 面接データ: 面接中の音声、その文字起こし、ホワイトボードの内容。音声はブラウザから OpenAI 社の API に直接送信され、応答の生成に利用されます。文字起こしとボード内容は評価レポートの生成のために OpenAI 社の API に送信されます。運営者のサーバー・データベースには面接の音声・会話内容を保存しません（評価レポートはユーザーのブラウザ上にのみ表示されます）。

2. 利用目的

  1. 本サービスの提供（認証、チケット管理、面接の実施、評価レポートの生成）
  2. 決済の処理および購入履歴の管理
  3. 不正利用の防止、お問い合わせへの対応

3. 第三者への提供・委託

運営者は、法令に基づく場合を除き、本人の同意なく個人情報を第三者に提供しません。本サービスの提供にあたり、以下の事業者に処理を委託しています。

  • Supabase Inc.（認証・データベース）
  • Stripe, Inc.（決済処理）
  • OpenAI（音声対話・評価レポートの生成）
  • Vercel Inc.（ホスティング）

これらの事業者のサーバーは国外に所在する場合があります。各社のプライバシーポリシーもあわせてご確認ください。

4. Cookie 等の利用

本サービスは、ログイン状態の維持のためにブラウザのローカルストレージを使用します。広告目的のトラッキングは行っていません。

5. 開示・訂正・削除の請求

ユーザーは、自己の個人情報の開示・訂正・利用停止・削除（アカウントの削除を含む）を請求できます。下記の連絡先までご連絡ください。合理的な期間内に対応します。

6. 改定

本ポリシーは必要に応じて改定されます。重要な変更がある場合は本サービス上で告知します。
